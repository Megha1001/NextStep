1. Euclidean algorithm -> To find GCD
    Let 'b' be smaller than 'a'
    gcd(a, b) = gcd(a-b, b)

    why ?
    Let 'g' be GCD of 'a' and 'b'
    a = gx, b = gy and gcd(x, y) = 1

    (a-b) = g(x-y)

2. Optimized Version of Euclidean algorithm
    gcd(int a, int b){
        if(b==0){
            return a;
        }
        return gcd(b, a%b)
    }

3. LCM of two numbers
    a*b = gcd(a,b) * lcm(a,b)
