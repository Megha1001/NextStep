## Pattern Cheat Sheet (Quick Reference)

| Pattern | Template | Key LinkedIn Problems |
|---------|---------|----------------------|
| **HashMap + DLL** | Node(key,val,prev,next), addToTail(), removeNode() | LRU Cache, All O'one, Max Stack |
| **Sliding Window** | `while(r<n){ expand; while(invalid) shrink left; update }` | Max Consecutive Ones III, Min Window Substring |
| **Kadane** | `maxHere = max(num, maxHere + num)` | Max Subarray, Max Product Subarray |
| **Tree DFS** | Base: null→return; recurse left,right; process | Validate BST, LCA, Find Leaves, Diameter |
| **Binary Search** | `lo,hi,mid=lo+(hi-lo)/2` | Rotated Array, 2D Matrix, K Closest |
| **Graph BFS** | Queue + visited + level tracking | Number of Islands, Word Ladder, Clone Graph |
| **Backtracking** | Choose → Explore → Unchoose | Permutations, Letter Combinations |
| **Stack** | Push operand, pop on operator/closer | RPN, Exclusive Time, Valid Parentheses |
